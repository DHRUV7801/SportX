# SportX --- E-commerce Frontend

A modern, responsive **sports gear e-commerce frontend** built using
**HTML, CSS, and JavaScript**.

## 🚀 Features

-   🛒 Add to Cart with dynamic updates
-   ⚡ Quick Add functionality
-   📦 Product listing with ratings & pricing
-   📱 Fully responsive design
-   🎯 Popup discount modal
-   ⏳ Countdown timer for offers
-   ❤️ Wishlist toggle
-   📂 Category-based browsing

## 🧱 Tech Stack

-   HTML5
-   CSS3 (Custom styling + animations)
-   Vanilla JavaScript (No frameworks)

## 📁 Project Structure

    ├── index.html     # Main UI layout
    ├── styles.css     # Styling and responsiveness
    ├── main.js        # App logic (products, cart, interactions)
    └── images/        # Assets (not included here)

## 📌 How to Run

1.  Download or clone the project
2.  Keep all files in the same folder
3.  Open `index.html` in your browser

## ⚙️ Key Functionalities

### 1. Product Rendering

Products are dynamically rendered using JavaScript.

### 2. Cart System

-   Add/remove items
-   Update quantity
-   Auto total calculation

### 3. UI Interactions

-   Popup modal after 3.5 seconds
-   Mobile navigation menu
-   Smooth scrolling
